# Python Mini Game

Pune fișierul tău Python (`main.py`, `joc.py`, `app.py` sau `game.py`) în acest repo.

## Cum rulezi (varianta rapidă, GitHub Codespaces)
1. Deschide repo-ul pe GitHub.  
2. Click `Code` → `Codespaces` → `Create codespace on main`.  
3. Codespace-ul va porni și va rula automat `run.sh` (vezi `.devcontainer/devcontainer.json`).  
4. Vezi consola din Codespaces — acolo apar print-urile jocului.
